Algoritmo di Dijkstra

1 Algoritmi preliminari

/*
G = grafo
s = nodo sorgente
*/
INIT_SS(G, s){
    for each u in V[G]{
        d[u] = +inf
        π[u] = NIL
    }
    d[s] = 0
}

/*
v = nodo di arrivo
u = nodo precedente a v (?)
w(u, v) = peso dell'arco tra u e v
*/
RELAX(u, v, w(u, v)){
    if(d[v] > d[u] + w(u, v)){
        d[v] = d[u] + w(u, v)
        π[v] = u
    }
}

2 Proprietà preliminari

2.1 Proprietà del limite inferiore

Se si inizializza con INITSS, per qualunque sequenza di RELAX si ha:

δ(s, u) <= d[u] — ∀ u ∈ V

Inoltre, se dopo una RELAX

δ(s, u) = d[u]

allora il valore di d[u] non potrà cambiare mai più

2.2 Proprietà dell’assenza di cammino

Se non esiste un cammino tra s e u

δ(s, u) = +inf

e allora dopo la INITSS si ha

d[u] = δ(s, u)

2.3 Proprietà del grafo dei predecessori

Se alla fine di un algoritmo che usa RELAX si ha:

d[u] = δ(s, u)

Allora Gπ è un albero di cammini minimi

2.4 Distanza triangolare

Dato (u, v) ∈ E

δ(s, v) = δ(s, u) + w(u, v)

3 Algoritmo

/*
G = grafo
w = funzione peso
s = nodo sorgente
*/
DIJKSTRA(G, w, s){
    INIT_SS(G, s)
    Q <- V[G]
    S <- NIL

    while(Q != NIL){
        u <- EXTRACT_MIN(Q)
        S <- S U {u}

        for each V in ADJ[u]
            RELAX(u, v, w(u, v))
    }

    return (d, Gπ)
}

4 Complessità

4.1 Implementazione con Q array lineare

n (inizializzazione) + n2 (extractmin) + m (relax, m = out-degree)

= O(n + n2 + m) -> O(n^2)

4.2 Implementazione con Q heap binario

n (inizializzazione) + nlogn (extractmin) + mlogn (relax + ribilanciamento dell’albero)

= O(n + nlogn + mlogn) -> O(mlogn)

(In base a cosa m > n ??)

5 Dimostrazione della correttezza

Sempre vera dopo la INITSS:

Se u != s –> d[u] = +inf –> δ(s, s) = -inf
Se u == s –> d[s] = 0 –> δ(s, s) = 0

δ(s, u) = d[u]

PER ASSURDO: Esiste v ∈ V tale che dopo una RELAX d[v] < δ(s, v)

Dopo la RELAX(u, v, w(u, v)) abbiamo che
d[u] + w(u, v) = d[v]
di conseguenza d[u] < δ(s, v) per ipotesi
e a sua volta d[u] = δ(s, u) + w(u, v)
e di conseguenza d[u] < δ(s, u), che è assurdo
C.V.D
